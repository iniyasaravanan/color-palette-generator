# Color Palette Generator

Color Palette Generator is a web application that allows users to create, customize, and export color palettes. It provides an interactive interface for exploring color combinations, adjusting color values, and ensuring accessibility compliance.

## Overview

This application enables users to generate color palettes from a base color or randomly, explore different color harmonies, and preview colors in multiple formats (HEX, RGB, HSL). It also includes tools for exporting palettes and checking accessibility between color combinations.

## Features

- Generate color palettes from a base color or randomly  
- Support for multiple color harmony modes  
- View colors in HEX, RGB, and HSL formats  
- Save and export palettes  
- Gradient preview and color sorting options  
- Accessibility checker for foreground/background contrast  
- Clean, responsive UI with dark/light mode support  

## Tech Stack

- **TypeScript**
- **React**
- **Tailwind CSS**
- **Vite**

## Libraries Used

- **tinycolor2** – color generation and manipulation  
- **lucide-react** – icons  
- **Material UI** – UI components
